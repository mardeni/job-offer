## Como Utilizar

Os códigos devem ser enviados via URL para os candidatos:

**Formato do Link:**
```
http://seu-dominio.com/?codigo-recrutador=CM2024
```

**Exemplo:**
```
http://localhost:3000/?codigo-recrutador=LEG001
```

---

## Observações Importantes

1. ✅ Todos os códigos possuem exatamente 6 caracteres
2. ✅ São case-insensitive (CM2024 = cm2024)
3. ⚠️ **Cada código pode ser usado apenas UMA vez** - após submissão, o código se torna inválido
4. ⚠️ O sistema só aceita acesso via dispositivos móveis
5. ⚠️ A geolocalização deve estar ativada no dispositivo
6. ⚠️ O usuário deve autorizar o uso da localização antes de submeter

---
